# 電腦與物理 · TD3 × MuJoCo Ant-v5 — 實驗數據網站

四足步態強化學習（TD3）期末專題的完整數據附錄與程式碼瀏覽，補充簡報未收錄的細節。

## 內容

| 路徑 | 說明 |
|---|---|
| `index.html` | 主網頁：版本演進、九大步態指標、TensorBoard 分析、各版本 reward 設定與數據、成果影片 |
| `code.html` | 程式碼專區：頁內檔案瀏覽器（語法高亮 + Markdown 渲染，所有檔案內嵌） |
